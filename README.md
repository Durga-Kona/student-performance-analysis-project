# student-performance-analysis-project

📊 Student Performance Analysis
Project Overview
This project analyzes a student performance dataset to understand how study habits and other factors influence academic results.
The analysis was performed using Python and visualized using different charts.

Objective
The main goals of this project are:
Explore the relationship between study time and student scores
Identify patterns in student performance
Visualize correlations between different factors affecting academic results

Tools Used
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook

Dataset
The dataset contains student information including:
Study time
Absences
Final grades
Demographic information
Analysis Performed
The following analysis was conducted:
Data exploration and summary statistics
Scatter plot analysis of study time vs final grade
Correlation heatmap of dataset variables

Key Insights
Students who spend more time studying generally achieve higher scores.
Attendance and study habits play an important role in academic performance.
Some variables show strong correlations with final grades.

Project Files
student_performance.ipynb – Jupyter notebook containing analysis
student_data.csv – Dataset used in the project
